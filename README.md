# Test task

Client and server side web application. You can read about requirements [here](REQUIREMENTS.md).

## Before started

For both client and server side app requires [Node.js](https://nodejs.org/) v12+ to run.
For working Google Maps, you need to obtain [API key](https://developers.google.com/maps/documentation/javascript/get-api-key).
Also you need [Google OAuth Credentials](https://support.google.com/cloud/answer/6158849?hl=en) to make authorization work.

## Setup instructions

To setup application, you need to setup both client and server side.
You can read relevant instructions in their folders:

- [Client side](client/README.md)
- [Server side](server/README.md)
